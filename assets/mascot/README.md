Place your mascot reference images here when using `brand_mascot` with `Veo`.

Recommended set:
- `front.png`: clean front-facing view
- `three_quarter.png`: 3/4 angle view
- `profile.png`: side or alternate expression

Tips:
- Keep the same shape, colors, eyes, and mouth in every image.
- Use simple backgrounds or transparent PNGs when possible.
- Export high-resolution images with the mascot clearly visible.
- Use the prompt pack in `assets/mascot/PROMPTS.md` to generate the starter set.

Then point `shared/brand_config.yml` `brand_mascot.reference_images` to these files.
