# Jess Candle Reference Prompts

Use these prompts to generate the 3 core reference images for the Jess Trading mascot before testing `brand_mascot` with `Veo`.

Recommended generation settings:
- Aspect ratio: `1:1`
- Background: plain dark charcoal or transparent if the model supports it
- Framing: full body, centered, character clearly visible
- Style target: premium 3D cartoon mascot, fintech brand asset, not children's illustration
- Keep the same seed/style preset if your image model allows it

Core character spec to preserve in every image:
- Humanized neon-green trading candlestick mascot
- Candle body is the actual torso and main silhouette
- Rounded cartoon eyes, expressive eyebrows, clean friendly smile
- Small simple arms with gloved hands
- Small simple legs or base stance, but secondary to the candle body
- Fintech look: polished, modern, confident, brand-safe
- Must feel like the spokesperson of a premium algorithmic trading company
- Never turn into a human trader, superhero, wax candle, or generic emoji

## Prompt 1: Front View

```text
Create a premium 3D cartoon mascot character for a fintech trading brand: a humanized neon-green trading candlestick named Jess Candle. The character is a bright green market candle with the candle body as the main torso and silhouette, not a human in a costume. Give it clean expressive cartoon eyes, subtle eyebrows, a confident friendly smile, small elegant arms with simple gloves, and minimal legs or a grounded stance. The design should feel polished, memorable, and brand-ready, like a studio-designed mascot for a premium algorithmic trading company.

Show the character in a full-body front-facing neutral pose on a simple dark charcoal studio background. Keep the mascot centered, clearly readable, symmetrical, and easy to use as a reference image. Premium 3D cartoon rendering, soft studio lighting, crisp edges, clean reflections, high detail, modern fintech aesthetic, no clutter, no props, no text, no watermark.
```

## Prompt 2: Three-Quarter View

```text
Create the same exact mascot character as a consistent turn-around reference: Jess Candle, a humanized neon-green trading candlestick for a premium fintech brand. The candle body must remain the primary silhouette, with the same face placement, proportions, expression language, bright green color, polished cartoon material, small arms with gloves, and clean mascot styling. The character should look charismatic, clever, and trustworthy, like the spokesperson of an algorithmic trading company.

Show the character in a full-body three-quarter view facing slightly to the left, with a confident open posture and subtle hand gesture as if explaining a trading concept. Use a simple dark charcoal studio background. Premium 3D cartoon render, cinematic studio lighting, clean shadows, high consistency with the front view, no props, no text, no watermark.
```

## Prompt 3: Profile View

```text
Create the same exact mascot character again as a side-profile reference image: Jess Candle, a neon-green humanized trading candlestick mascot with a cartoon face, small expressive arms, polished 3D cartoon styling, and a premium fintech brand look. Preserve the exact same shape language, facial features, green color, and proportions from the previous views. The character must still read immediately as a trading candle, not a human or toy.

```
